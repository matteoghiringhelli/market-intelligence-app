# Market Intelligence App

Applicazione web a finalità esclusivamente informativa ed educativa per consultare dati descrittivi su titoli azionari, peer group, pattern tecnici, disclosure pubbliche e qualità dati.

## Cosa NON fa

Questa app non fornisce:

- consulenza finanziaria;
- segnali di trading;
- raccomandazioni buy/sell/hold;
- indicazioni operative personalizzate;
- previsioni certe.

## Stack

- GitHub repository
- GitHub Codespaces per sviluppo cloud
- Vercel per deploy
- Frontend Vanilla JavaScript con Vite
- Dati mock iniziali salvati nel repository
