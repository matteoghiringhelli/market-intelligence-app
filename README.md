# Market Intelligence App

Applicazione web a finalità esclusivamente informativa ed educativa per analizzare dati di mercato azionario.

Questa app non fornisce consulenza finanziaria, segnali di trading, raccomandazioni buy/sell/hold o indicazioni operative personalizzate.

## Stack previsto

- GitHub
- GitHub Codespaces
- Vercel
- Vite
- Vanilla JavaScript
